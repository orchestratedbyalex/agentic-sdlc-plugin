What it does for you

A disciplined process, run on autopilot

You bring the idea or the codebase. The plugin brings the rigor of a full software team — and keeps the receipts.

Idea → shipped code

It plans what to build, designs how, writes the code and the tests, proves it works, and prepares the release — in order, nothing skipped.

Checked at every step

The agent that writes something is never the one that approves it. An independent reviewer inspects each phase before the work moves forward.

Writes the docs for you

Requirements, user stories, architecture decisions — all generated and saved in your repository as you go, not as an afterthought.

Stop and resume anytime

Every step is saved. Close your laptop mid-phase and the wizard picks up at the exact agent where you left off — even on another machine.

The journey

Seven phases, start to finish

Each phase hands off to the next only after passing a quality check. After shipping, what's learned in Operate feeds straight back into the next round.

Under the hood

How each phase keeps quality high

The same simple loop runs inside every phase. It's why long runs don't drift and why a mistake gets caught instead of shipped.

01 — DO

Specialists do the work

Focused agents each handle one job — a requirements writer, an architect, a security checker, a test author.

02 — CHECK

An independent reviewer gates it

A separate, read-only reviewer inspects the output. It can't edit the work — only judge it. Real builds are actually run.

03 — ROUTE

Pass moves on, fail goes back

Clean? Continue to the next phase. Problems? Sent back to be fixed and re-checked — automatically.

Get started

Install & run

No build step, no heavy dependencies. If you have Claude Code, you're a couple of commands away.

Check the requirements

Claude Code (CLI, desktop, web, or IDE) and Node.js 18+ (Node 22 LTS recommended). That's it — there's no npm install.

Add the marketplace & install

Inside Claude Code, run:

/plugin marketplace add orchestratedbyalex/agentic-sdlc-plugin
/plugin install agentic-sdlc@agentic-sdlc-marketplace

Run the wizard

In any repository:

/sdlc   # detects empty project, existing code, or a run to resume

It prints a seven-phase status board and routes you to the right starting point — scaffold a new project, set up an existing one, or continue where you stopped.

Where things live

The plugin runs the process — your repo keeps the record

A clean separation: the plugin stays generic and reusable; everything specific to your project is written into your own repository.

The plugin · "how"

Carries the process

The wizard, the 35 specialist agents, the phase playbooks, the quality gates. Installed once, used on any project. Nothing about your code lives here.

→
Your repository · "what"

Collects the evidence

Every artifact the run produces is saved in your project, so the record travels with your code:

  • requirements & user stories in docs/requirements/
  • architecture & decisions in docs/design/
  • operations notes in docs/operate/
  • resumable progress in sdlc-metadata.yml

You stay in control

The agents stage commits and suggest tags — they never commit, push, or publish for you. Anything that leaves your machine is your call. Git stays in your hands, always.
